# 使用 python or js 抓取 kobo 每週 99 元書單

1. 利用爬蟲工具抓取每週99元的書單。
2. 使用http請求，執行抓取資料。

## JS
### Installation
安裝套件
```bash
npm install express chromedriver selenium-webdriver
```

執行
```bash
node app.js
```


## python
### Installation
安裝套件
```bash
pip install selenium webdriver-manager Flask
```

執行
```bash
python scrape.py
```
